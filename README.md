# 🎮 RPG Terminal em C#

Um pequeno RPG de terminal desenvolvido em C# para praticar lógica de programação e estruturação de sistemas simples.

---

## ⚔️ Funcionalidades

- Sistema de batalha por turnos
- Ataque
- Cura
- Defesa
- Ataque crítico
- Evento aleatório ("Dado da sorte")
- Sistema de vida do jogador e do monstro

---

## 🛠️ Tecnologias utilizadas

- C#
- .NET
- Console Application

---

## 🚀 Como executar

1. Abra o projeto no terminal
2. Execute:

```bash
dotnet run
```

---

## 🧠 Objetivo do projeto

Esse projeto foi criado para praticar:
